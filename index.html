Primer Entregable

La simulacion sera un ecommerce de productos farmaceuticos deribados del cannabis. Por el momento tendra poco inventario.

  1. El sistema ofrece un menu de interaccion donde podra listar productos o generar una compra
  2. El sistema le pedira al cliente que ingrese el producto que busca. (prompt)
  3. Se buscará en el array de objetos que representará el inventario.
  4. Si se encuentra, el sistema mostrara detalles del producto y pedira una confirmacion. Si no se encuentra volvera a ofrecer el menu
  5. Luego el sistema preguntara si es usuario registrado del reprocann
  6. Si lo es recibira un descuento y lo mostrara por un "alert", si no lo es saltea este punto.
  7. Por ultimo el usuario debe ingresar sus datos para generar la facturacion y confirmar la compra.
  8. El sistema vuelve a preguntar si solicita algo mas

pd: Trate de evitar los metodos find() y filter() para darle un poquito de dificultad
